# 素材来源及处理说明

本宠物是基于《黑神话：悟空》官方封面中“天命人”无冠黑金甲造型制作的非官方同人 Codex 动画宠物。视觉参考来自游戏科学及 Steam 发布的官方宣传素材；本项目不打包或声称拥有原始宣传图。

宠物图集由生成式图像工具辅助重绘，并经过分帧、动作编排、透明背景处理和视觉校验。生成式重绘、Q 版化或重新编排不会消除原角色名称、造型及美术表达可能涉及的版权、商标和其他知识产权。

本项目与 Game Science（游戏科学）、《黑神话：悟空》及其相关权利人不存在授权、认可、赞助或合作关系。除非另行取得所需授权，建议仅作个人、本地、非商业使用。
